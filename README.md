# Solstice Landing Page

A modern, responsive landing page for Solstice, a boutique real estate community by Shea Homes located in Littleton, Colorado.

## Features

- **Responsive Design** - Mobile-first approach using Bootstrap 5
- **Hero Section** - Eye-catching hero with multiple image layouts
- **Collections Showcase** - Display of three home collections (Horizon, Reflection, Summit)
- **Contact Form** - Integrated contact form with Supabase backend
- **Gallery Section** - Image gallery showcasing the community
- **Modern Styling** - Custom SCSS with CSS variables for consistent theming
- **Form Validation** - Client-side validation with SweetAlert2 notifications
- **Accessibility** - SEO optimized and accessibility-focused

## Tech Stack

- **Frontend**: HTML5, Bootstrap 5, SCSS
- **Backend**: Vercel API Routes
- **Database**: Supabase
- **Notifications**: SweetAlert2
- **Deployment**: Vercel
